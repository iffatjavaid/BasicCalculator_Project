🧮 Basic Calculator
📌 Description
This is a console-based Basic Calculator developed in Python as part of the CodeAlpha Python Programming Internship.
The calculator allows users to perform various mathematical operations such as addition, subtraction, multiplication, division, modulus, floor division, and exponentiation. It also handles invalid operations and division by zero gracefully.
✨ Features
Addition, Subtraction, Multiplication
Division with zero-division handling
Modulus (%)
Floor Division (//)
Exponent (**)
User-friendly console interface
Multiple calculations without restarting
🛠 Technologies Used
Python 3

Variables
Data Types
Input/Output
Arithmetic Operators
Comparison Operators
If-Else Statements
While Loop
Type Casting (int, float)
Break Statement
